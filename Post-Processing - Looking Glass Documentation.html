Post-Processing

Post-processing effects are commonly used in 3D imaging technologies to enhance rendered imagery. These effects can greatly enhance the visual quality of Looking Glass content.
The following types of effects tend to have a big effects on content in the Looking Glass.
  • Bloom
  • Color Grading
  • Depth of Field
  • Ambient Occlusion
  • Vignette
Every rendering environment approaches post-processing differently. To learn how to apply these effects in the following environments, please consult the corresponding pages:
  • Unity Plugin Post Processing documentation
  • Unreal Plugin Post Processing documentation
